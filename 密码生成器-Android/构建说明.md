# 密码生成器 - Android APK 构建说明

## 项目说明

这是一个标准 Android 项目，使用 **WebView + 原生 Java** 构建。
- 应用内嵌 HTML，无需联网，完全离线运行
- 支持 Android 5.0（API 21）及以上
- 一键复制密码（调用系统剪贴板，并有 Toast 提示）

---

## 方法一：用 Android Studio 打开（推荐，5分钟出 APK）

### 前提
- 安装 [Android Studio](https://developer.android.com/studio)（内置 JDK 和 SDK，无需单独安装）

### 步骤

1. 打开 Android Studio → **File → Open**
2. 选择本项目根目录（`密码生成器-Android/`）
3. 等待 Gradle 同步完成（首次约 3-5 分钟，需联网下载 Gradle）
4. 菜单 **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. 构建完成后点提示中的 **locate** 找到 APK 文件
   - 路径：`app/build/outputs/apk/release/密码生成器-v1.0.apk`

6. 将 APK 传到手机，允许"安装未知来源应用"后即可安装

---

## 方法二：GitHub Actions 在线构建（免费，无需安装任何东西）

### 步骤

1. 注册/登录 [GitHub](https://github.com)
2. 创建新仓库（public 或 private 均可）
3. 上传本项目所有文件到仓库
4. 进入仓库 → **Actions** 标签页
5. 点击 **Build APK** workflow → **Run workflow**
6. 等待约 3-5 分钟构建完成
7. 点击构建记录 → **Artifacts** → 下载 `密码生成器APK`

---

## 方法三：命令行构建（已有 JDK 环境）

```bash
# Windows
gradlew.bat assembleRelease

# Linux/macOS
chmod +x gradlew
./gradlew assembleRelease
```

APK 输出路径：`app/build/outputs/apk/release/密码生成器-v1.0.apk`

## 手机安装注意事项

1. 将 APK 文件通过微信、QQ、USB 等方式传输到手机
2. 在文件管理器中点击 APK 文件
3. 如提示"禁止安装未知来源应用"，进入 **设置 → 安全 → 允许安装未知来源** 开启
4. 完成安装，桌面会出现"密码生成器"图标
